# 기능목록

### RandomNumberGenerator

- 랜덤으로 3자리수 생성 (서로 다른 수로 이루어져야 함)-> Numbers class 리턴 [o]

### NumberComparator

- 스트라이크, 볼, 낫싱 판단 로직 [o]

### Number

- 세 개의 수 가진 리스트 [o]
- size 3 외 예외처리 [o]
- 번호 중복 예외처리 [o]

### GameResult

- 스트라이크, 볼, 낫싱 개수 리턴 [o]
- 3스트라이크 인 경우 게임 재시작/종료 1 or 2 [o]
- 1 /2 아닐 시 예외처리 [0]

### inputView

- 3개 숫자 입력받기 [o]
